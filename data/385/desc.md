Cho hai số nguyên không âm $a$ và $b$. Hãy tính $a×b$.

## Dữ liệu vào:
- Dòng đầu chứa số $a$.
- Dòng sau chứa số $b$.

## Dữ liệu ra:
- Gồm một dòng duy nhất là kết quả của $a×b$.
## Ví dụ:
#### Dữ liệu vào:
```
123
4567
```

#### Dữ liệu ra:
```
561741
```

#### Dữ liệu vào:
```
1234567890
9879879876543219876
```

#### Dữ liệu ra:
```
12197382452637423456119381640
```

## Giới hạn:
- $0 ≤ a, b < 10^{1000} (a, b\text{ có không quá }1000\text{ chữ số})$
