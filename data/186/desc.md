Nhập $n$. Cho biết $n$ có phải số chính phương không?

## Dữ liệu vào:
- Số nguyên dương $n\ (n≤ 10^9)$.

## Dữ liệu ra:
- Ghi `YES` nếu $n$ là số chính phương, ngược lại ghi `NO`.

## Ví dụ:
### Dữ liệu vào:
```
36
```

### Dữ liệu ra:
```
YES
```

### Dữ liệu vào:
```
10
```

### Dữ liệu ra:
```
NO
```
