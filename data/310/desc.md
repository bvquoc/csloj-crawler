Tính tổng các số tự nhiên liên tiếp $S_n=1+2+⋯+n$.

## Dữ liệu vào:
- Một dòng duy nhất chứa số nguyên dương $n$.

## Dữ liệu ra:
- Ghi trên một dòng số nguyên dương $S_n$.

## Ví dụ:
### Dữ liệu vào:
```
2
```

### Dữ liệu ra:
```
3
```

### Dữ liệu vào:
```
10
```

### Dữ liệu ra:
```
55
```

## Giới hạn:
- Trong tất cả các test: $1≤n≤10^9$
