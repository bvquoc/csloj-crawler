Tính tổng bình phương các số tự nhiên liên tiếp ~S_n=1^2+2^2+⋯+n^2~

## Dữ liệu vào:
- Một dòng duy nhất chứa số nguyên dương ~n~

## Dữ liệu ra:
- Ghi trên một dòng số nguyên dương ~S_n~

## Ví dụ:
#### Dữ liệu vào:
```
2
```

#### Dữ liệu ra:
```
5
```

#### Dữ liệu vào:
```
10
```

#### Dữ liệu ra:
```
385
```

## Giới hạn:
- Trong tất cả các test: ~1≤n≤10^6~
